Cold-start orientation for an AI coding agent.

Run this at the beginning of any session before writing code.

## 1. Read the entry point

Read .ai/context/context_base.md for:
- Project overview, tech stack, key paths
- .ai folder structure and navigation guide
- Extended context table of contents

## 2. Check current focus

Read .ai/plans/_index.md to see:
- What plan is currently active
- Which steps are in progress or on deck

If a plan is active, read the referenced plan file for details.

## 3. Load relevant context

Based on the task at hand, read the relevant context_extended/ files:
- Working on AI players? Read ai_players.md
- Working on API? Read api_routes.md
- Working on frontend? Read frontend.md
- Working on game logic? Read game_engine.md
- Working on storage/replay? Read storage.md
- Working on scripts/tooling? Read scripts_tooling.md
- Need the big picture? Read architecture.md

Only read what you need — don't load everything.

## 4. Confirm orientation

Before writing any code, confirm you understand:
- The current architecture and patterns in use
- What is being worked on and why
- Where the relevant source files are
